Nexo PDV - Modo App Sem Barra de URL

Esta página explica como abrir o Nexo PDV em modo app sem a barra de URL, proporcionando uma experiência mais limpa e profissional.

Nota: Esta solução requer o Google Chrome instalado no computador.
Novo! Agora você pode personalizar o ícone dos atalhos com um ícone de carrinho de compras.

O que é o Modo App?

O Modo App é uma funcionalidade do Google Chrome que permite abrir sites como se fossem aplicativos nativos, sem mostrar a barra de URL e outros elementos do navegador. Isso proporciona uma experiência mais limpa e profissional para o usuário.

Novidade: Todos os scripts agora abrem o Chrome em tela cheia automaticamente, adaptando-se à resolução do seu monitor.

Opções de Instalação

Para Windows

Escolha uma das opções abaixo:

Opção 1: Versão para Produção

Esta versão é simples e abre diretamente o ambiente de produção.

Baixar Versão Produção

Opção 2: Versão para Desenvolvimento com Porta

Esta versão é simples como a de produção, mas permite configurar a porta para o ambiente de desenvolvimento local.

Baixar Versão Desenvolvimento Simples

Opção 3: Versão para Desenvolvimento Avançada

Esta versão permite configurar a porta para o ambiente de desenvolvimento local com opções adicionais.

Baixar Versão Desenvolvimento Avançada

Opção 4: Versão Completa

Esta versão permite escolher entre diferentes ambientes, mas pode apresentar problemas em alguns sistemas.

Baixar Versão Completa

Para Linux e macOS

Use o script shell abaixo:

Baixar Script Shell

Após baixar, torne o arquivo executável com o comando:

chmod +x abrir-nexo-pdv.sh

Como Usar

No Windows:

  1. Baixe um dos arquivos .bat acima
  2. Salve o arquivo em um local de fácil acesso (ex: Área de Trabalho)
  3. Dê um duplo clique no arquivo para executá-lo
  4. Se estiver usando a versão completa, selecione o ambiente desejado
  5. O Nexo PDV será aberto em modo app, sem a barra de URL

No Linux ou macOS:

  1. Baixe o arquivo .sh acima
  2. Abra o terminal e navegue até a pasta onde o arquivo foi salvo
  3. Torne o arquivo executável com o comando: chmod +x abrir-nexo-pdv.sh
  4. Execute o arquivo com o comando: ./abrir-nexo-pdv.sh
  5. O Nexo PDV será aberto em modo app, sem a barra de URL

Criando um Atalho na Área de Trabalho

No Windows:

  1. Clique com o botão direito no arquivo .bat
  2. Selecione "Criar Atalho"
  3. Mova o atalho para a Área de Trabalho
  4. Para personalizar o ícone, clique com o botão direito no atalho, selecione "Propriedades" e depois "Alterar ícone"
  5. Novo! Clique aqui para obter um ícone de carrinho de compras para seus atalhos

No Linux:

  1. Crie um arquivo .desktop na pasta ~/.local/share/applications/
  2. Adicione o seguinte conteúdo (ajustando os caminhos conforme necessário):
    3. [Desktop Entry]
    Name=Nexo PDV
    Exec=/caminho/para/abrir-nexo-pdv.sh
    Icon=/caminho/para/icone.png
    Type=Application
    Categories=Office;

Perguntas Frequentes

O Chrome não é encontrado automaticamente

Se o script não encontrar o Chrome automaticamente, você pode editar o arquivo .bat ou .sh e ajustar o caminho para o executável do Chrome no seu sistema.

Quero usar um ambiente diferente

Na versão completa para Windows e Linux/macOS, você pode selecionar diferentes ambientes (produção, desenvolvimento, etc.). Para o ambiente de desenvolvimento, o script perguntará em qual porta o projeto está rodando. Na versão simples, você pode editar o arquivo e alterar a URL para o ambiente desejado.

O que fazer se aparecer um aviso de segurança?

Alguns sistemas podem mostrar um aviso de segurança ao executar arquivos .bat. Isso é normal. Você pode clicar em "Mais informações" e depois em "Executar assim mesmo" para prosseguir.

Porta personalizada para ambiente de desenvolvimento

Se você estiver usando uma das versões para desenvolvimento (abrir-nexo-pdv-porta.bat ou abrir-nexo-pdv-dev.bat), o script perguntará em qual porta o projeto está rodando. Por padrão, a porta é 5173, mas você pode especificar outra porta se necessário.

Tela Cheia: Todos os scripts agora abrem o Chrome em tela cheia automaticamente, proporcionando uma experiência mais imersiva.

Recomendação: Para melhor experiência, use a versão específica para o ambiente que você precisa:

  • Use abrir-nexo-pdv-simples.bat para acessar o ambiente de produção
  • Use abrir-nexo-pdv-porta.bat para acessar o ambiente de desenvolvimento local (versão simples)
  • Use abrir-nexo-pdv-dev.bat para acessar o ambiente de desenvolvimento local (versão avançada)

Dica: Você pode criar um atalho para este arquivo na Área de Trabalho ou na barra de tarefas para acessar o Nexo PDV rapidamente.
