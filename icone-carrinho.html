Ícone de Carrinho de Compras para Nexo PDV

Esta página fornece um ícone de carrinho de compras para ser usado com os atalhos dos arquivos .bat do Nexo PDV.

Como Usar Este Ícone

Passo 1: Salvar o Ícone

Clique com o botão direito na imagem do carrinho acima e selecione "Salvar imagem como..." para salvar o ícone em seu computador.

Alternativamente, você pode usar o botão abaixo para baixar o ícone em formato SVG:

Baixar Ícone SVG

Passo 2: Criar Atalhos para os Arquivos .bat

  1. Clique com o botão direito no arquivo .bat (por exemplo, abrir-nexo-pdv-porta.bat)
  2. Selecione "Criar Atalho"
  3. Mova o atalho criado para a Área de Trabalho ou outro local de sua preferência

Passo 3: Alterar o Ícone do Atalho

  1. Clique com o botão direito no atalho criado
  2. Selecione "Propriedades"
  3. Na janela de propriedades, clique no botão "Alterar ícone"
  4. Clique em "Procurar" e navegue até o local onde você salvou o ícone
  5. Selecione o ícone e clique em "Abrir"
  6. Clique em "OK" e depois em "Aplicar" para salvar as alterações

Nota: Se você tiver problemas para usar o ícone SVG, você pode convertê-lo para o formato ICO usando ferramentas online como Convertio ou Aconvert.
